# Mesto
Описание проекта: Место - это интерактивная страница, на которой пользователи могут делиться фотографиями, удалять их и ставить лайки.

Функционал:
- Редактирование профиля,
- добавление/удаление фотографий,
- лайки/дизлайки,

Стек:
- JavaScript, 
- CSS,
- HTML.
